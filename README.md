# Rhetorical Scenarios Dataset

This repository contains the full conversation transcripts and annotated dataset used in the paper, *Reversing Constraint Collapse as a Solution to the Constraint Problem.* It includes all scenarios, researcher-annotated elicitation moves and constraint types, and the variable dictionary defining each column.

## Repository Structure

## Files

### `/dataset/`
- **rhetorical_scenarios.csv**  
  The full annotated dataset, including elicitation moves, constraint types, question types, and outcome summaries. Best viewed locally for full text visibility.

- **rhetorical_variable_dictionary.csv**  
  Definitions for each column in the dataset.

### `/transcripts/`
- **scenarios_transcript.md**  
  The complete turn-by-turn transcript for all five scenarios. Formatted with speaker labels (U0, R0, etc.) and including no identifying information.

## Annotation Overview

Annotations include:
- **elicitation_move** — strategy used by Rhetorical to surface constraints.
- **constraint_types** — researcher-coded categories of constraints elicited.
- **question_type** — type of question used for elicitation.
- **outcome** — summary of interpretive moves and surfaced constraints. 

See the variable dictionary for full definitions.
